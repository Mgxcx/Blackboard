# Blackboard
Projet réalisé lors de la 4ème semaine de formation du 2 au 4 Novembre. Blackboard est un back-office de commandes de trottinettes électriques. Ce dashboard permet de suivre les commandes de trottinettes effectuées par les clients ou encore les messages que les clients envoient. Il permet également de visualiser les tâches en attente.
